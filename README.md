# 📦 P&P Soluciones - Sistema ERP y Catálogo Digital

Bienvenidos al repositorio oficial de **P&P Soluciones**. 
Somos especialistas en Papelería, Material de Oficinas, Centro de Copiado y Servicio Técnico especializado en Computación y Redes.

## 🚀 Arquitectura Tecnológica
Este proyecto aloja la **Aplicación Web Progresiva (PWA)** pública para nuestros clientes, vinculada en tiempo real a nuestro **Backoffice ERP (P&P Admin)** impulsado por PostgreSQL y Supabase. 
Gestionamos de forma integral nuestro inventario, facturación POS, RMAs y tesorería en la nube.
